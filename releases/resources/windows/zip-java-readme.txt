Requirements
------------
This is a Windows package of @title@, including a Java Runtime Environment.
Since all the requirements are included in the package, this should run on
Windows Vista, 7, or any newer version.


Running @title@
-----------
To start @title@, find where you extracted the ZIP file, then double-click the
JES.exe file (its icon has the face of a python on it).
