@title@ is a development environment designed for Media Computation. It allows students to use the Python programming language (specifically, Jython, which is a version of Python implemented in Java) to manipulate images, sounds, and videos.

JES is incorporated in “Introduction to Computing and Programming in Python: A Multimedia Approach,” by Mark Guzdial and Barbara Ericson. Dr. Guzdial is the project leader, and the project has been worked on by many people over the years.

While we don't have the resources for extensive compatibility testing, we believe that JES will work on any currently-supported OS release and CPU architecture, as long as Java is available.

We also provide a testing kit, for testing or debugging JES: @basename@-@release@-testkit.zip. It includes copies of JES's unit tests, demo scripts, and a test plan. (The testing kit is cross-platform.)
