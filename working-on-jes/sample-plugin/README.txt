JES Sample Plugin
=================
This is a plugin for JES that makes a state-of-the-art "Hello, world"
library available for use.


Functions
---------
The only function provided is `greet`. To use it, first you must:

    >>> from greetings import greet

Then, you call it with a name:

    >>> greet("Michael")
    'Hello, Michael!'


Installation
------------
This plugin requires JES 5. To install this plugin:

1.  Open JES.
2.  Open the "File" menu and select "Install Plugin..."
3.  A file chooser will appear.
    Select the `sample-plugin.jar` file, and click "Open."
4.  A dialog box will appear to make sure you want to install the plugin.
    Answer "Yes."
5.  Restart JES for the plugin to load.
